NETIPFLIX - Samsung TV App
===========================

Este é um Tizen Web App para Samsung Smart TVs.

INSTALAÇÃO RÁPIDA:
1. Execute build-tizen.bat na pasta raiz
2. Copie o arquivo .wgt gerado para pendrive/userwidget/
3. Conecte o pendrive na TV Samsung
4. Confirme a instalação

Para instruções completas, veja: README-TIZEN.md

URL padrão do app: https://netipflix.pages.dev
Para alterar, edite index.html (linha const APP_URL)
